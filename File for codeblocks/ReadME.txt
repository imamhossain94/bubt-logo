1. Copy graphics.h and winbgim.h to your C:\\codeblocks\MinGW\include\    folder
2. Copy libbgi.a to C:\\codeblocks\MinGW\lib folder
3. Open codeblocks
4. Go to settings\Compiler\Linker settings
5. add libbgi.a in left white bpx
6. Click on the other white space box in right side.
7. Paste the content of "Linker Settings" file
8. Press ok then close codeblocks.
9. Re open codeblocks.
10. Run analog_clock.cpp & compile to test graphic
